# OpenAI Build Week submission checklist

- [ ] Select **Education** as the category.
- [ ] Publish the MIT-licensed repository or grant the required judge accounts access if the repository becomes private.
- [ ] Confirm `README.md` contains setup instructions, test commands, and Codex/GPT-5.6 disclosure.
- [ ] Run `npm run validate:challenges`, `npm test`, `npm run lint`, and `npm run build` from a clean clone.
- [ ] Upload a public, voiced YouTube demo under three minutes using `demo-script.md`.
- [ ] Capture the `/feedback` session ID where the core build work occurred and enter it in Devpost.
- [ ] Add any judge-specific local-run instructions to the Devpost private instructions field.

Do not submit the project, upload a video, create a Devpost project, or publish a repository without explicit authorization from the project owner.
